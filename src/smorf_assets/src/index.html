about smorf

what is smorf?

smorf is a simple dao factory for smo. the smorf main idea is to make a simple to use tool to create and manage simple dao. anybody as an admin of new smo will be able to create setup and manage simple blockchain backed dao. smorf initialy targets small comunities up to 1000 users.

what is smo?

smo is a self managed organisations. any kind of professionals can start the self driven comunity with common budget and voting driven desigion making
